# Classic Board Game Demo

## Overview
This project implements a GUI-based Tic Tac Toe game using Python's `tkinter` library.

## main.py
The main entry point of the application that creates an interactive Tic Tac Toe game interface.

### Features
- **GUI Interface**: Built with tkinter for cross-platform compatibility
- **Game Board**: 3x3 grid for standard Tic Tac Toe gameplay
- **Player vs Player**: Two players can alternate turns (X and O)
- **Game Logic**: Automatically detects win conditions and draws
- **Reset Button**: Restart the game at any time

### How to Run
```bash
python main.py
```

### Dependencies
- Python 3.x
- tkinter (included with most Python installations)

### Game Instructions
1. Run the application
2. Click on empty squares to make your move
3. Players alternate between X and O
4. First player to get three in a row wins
5. Use the reset button to start a new game
